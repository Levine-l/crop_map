Crop-profile zones give spatial context to SAR added value

The k=3 crop-profile solution is used for interpretation because it separates meaningful agricultural profiles while keeping enough test samples per zone. Zone sample counts are held-out CROME test samples assigned to each crop-profile zone; grid-cell counts are reported separately.
